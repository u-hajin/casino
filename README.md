# Casino game with C programming

You can play Casino game with this program. There are 6 games.
- Blackjack
- Horse racing
- Roulette
- Russian roulette
- Slot machine
- Tai Sai

## Contribution

- Opening, Ending animation
- Slot machine
- Tai Sai
